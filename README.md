## Idées

### BPR

$$
t = t_0 \cdot ( 1 + \alpha \frac{N}{C}^{\beta} )
$$
- $N$: Number of people in cell
- $C$: Capacity of cell

Two scenarios possible:
1. Fluide : $\alpha = 0.5$, $\beta = 2$ 
2. Panique : $\alpha = 1$, $\beta = 4$
